# Meetnow


A professional platform designed for seamless video meetings and real-time collaboration.

## Features
- **Real-time Video Meetings:** High-quality, low-latency video and audio communication.
- **Secure Authentication:** User registration and login flows with robust security.
- **Meeting Scheduling:** Create and manage upcoming meetings easily.
- **Interactive Chat:** Real-time messaging during meetings.
- **Meeting History & Analytics:** Track past meetings and user engagement.
- **Responsive Design:** A beautiful, modern interface built for any device.

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS, Radix UI
- **Backend:** Node.js, Express, Socket.io
- **Database:** PostgreSQL (via Drizzle ORM)
- **Validation:** Zod
- **API Client:** Orval

## Local Setup Instructions

### Prerequisites
- Node.js (v20+ recommended)
- pnpm

### Installation & Running Locally
1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Connect-Hub
   ```
2. **Install dependencies:**
   ```bash
   pnpm install
   ```
3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and configure the required variables (e.g., database connection string, session secrets).
4. **Run the project locally:**
   ```bash
   pnpm run dev
   ```
